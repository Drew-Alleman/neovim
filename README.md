## Neovim Config
My config for Neovim


### Instaling Neovim
```
sudo apt update
sudo apt upgrade
sudo apt install neovim
```
### Binding Neovim to ```v```
```
echo "alias v='vim'" >> ~/.bashrc
source ~/.bashrc
```
